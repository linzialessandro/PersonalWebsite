# Terminal Window Component

The `TerminalWindow` component is a stylized UI element that mimics the look and feel of a macOS or Linux terminal window.

## Location

Located at `src/components/TerminalWindow.jsx`.

## Features

- Provides a "window" frame with standard traffic-light buttons (close, minimize, maximize).
- Often uses `framer-motion` for reveal or typing animations.
- Capable of rendering children as "console output" for a developer-centric aesthetic.
